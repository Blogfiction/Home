# Carpeta de Audio

Coloca aquí tu archivo de música de fondo.

## Instrucciones

1. Coloca tu archivo de música en esta carpeta con el nombre `background-music.mp3` o `background-music.ogg`
2. El componente de música de fondo buscará automáticamente estos archivos
3. Formatos soportados:
   - MP3 (recomendado para máxima compatibilidad)
   - OGG (mejor compresión, pero menos compatible)

## Ubicación del archivo

El componente busca el archivo en:
- `./audio/background-music.mp3`
- `./audio/background-music.ogg`

Si usas otro nombre o ubicación, edita el archivo `components/BackgroundMusic.html` y actualiza las rutas en el elemento `<audio>`.
